## 🌟 项目简介

**龙门星脉求索虾**是一套运行在 OpenClaw 平台上的 AI 学术关系智能体系统。

它不是一只帮你机械搜论文的虾，而是一只真正懂学术关系、懂科研成长、懂合作时机的智能求索虾。它把公开学术世界中的人物、论文、动态、关系与机会，组织成一张可计算的**学术星脉图**，让研究生不只看见信息，更看见方向、合作与未来。

### 🔭 能力1 · 导师向上管理 `MentorOS`

> *"研究生最大的风险，往往不是不会做研究，而是没有用导师喜欢的方式做研究。"*

**数据采集层（多源并行）：**
- 📡 微信公众号 · 小红书 · 百度新闻 · Google Scholar — 实时动态与公开言论
- 📚 arXiv · DBLP · 百度学术 — 近期论文与研究轨迹
- 🏛️ 机构官网 · 学术主页 — 职位、项目、团队信息

**画像构建维度：**

| 维度 | 内容 |
|------|------|
| 🏢 基本信息 | 所在单位、职位、研究组、联系方式 |
| 📈 研究方向演变 | 5年时间轴分析，识别方向是收敛/扩张/转型 |
| 🔥 近期关注点 | 6个月内最新论文、公开发言、会议参与 |
| 🎭 性格与风格 | 从公开言行推断沟通偏好、学生偏好类型 |
| 🌿 生活习性 | 作息规律、兴趣爱好、社交风格 |
| 🕸️ 关系网络 | 导师的导师、核心合作者、师兄弟、机构资源 |
| 💬 沟通策略 | 最佳汇报时机、话术模板、idea 切入点 |
| 🎯 资源机会 | 可参与的项目线索、可争取的资源与支持 |

**输出：** 一份可操作的**导师行动手册**，让每次汇报有备而来，每个 idea 有的放矢。

---

### 👥 能力2 · 同门近期画像 `PeerGraph`

> *"明明身边就有人能合作，但彼此不知道对方此刻最擅长什么、最需要什么。"*

**数据采集层（多源并行）：**
- 📡 微信公众号 · 小红书 — 生活侧写与近期动态
- 📚 arXiv（逐篇核实作者列表）— 防止同名混淆
- 🎓 学校官网 · LinkedIn · 学术主页 — 背景核实

**画像构建维度：**

| 维度 | 内容 |
|------|------|
| 🎓 学术背景 | 本科/博士院校、专业方向、入学年份 |
| 🗺️ 研究演变 | 近6个月最新 + 5年历史，识别方向变化 |
| ⚙️ 技能图谱 | 擅长方向、工具栈、方法论，广度型 vs 深度型 |
| 🏆 近期成果 | 最新论文、项目进展、获奖情况 |
| 🎭 性格与兴趣 | 性格特点、生活习性、兴趣爱好 |
| 🤝 共同点分析 | 与用户的研究交叉点、共同话题、破冰切口 |
| 💡 互补价值 | 技能互补点、合作潜力评估、接近方式建议 |

**特色：** 多人**并行子 Agent 同时生成**，两人画像总时间 ≈ 单人，效率翻倍。

---

### 🚀 能力3 · 合作共赢未来 `Co-Lab Forge`

> *"在理解关系网络之后，怎样把'我想做研究'转成一个可以推进、可以分工、可以共赢的合作方案？"*

**输入：** 能力1的导师基本面 × 能力2的同门画像 × 用户自身方向

**输出维度：**

| 维度 | 内容 |
|------|------|
| 💡 Research Ideas | 3-5个具体可行方向，基于导师关注点 × 同门技能 × 用户方向的交叉点 |
| 📊 分工方案 | 谁做实验、谁做系统、谁做理论、谁对接导师，基于技能互补最优分工 |
| 🎯 沟通策略 | 与导师沟通的最佳时机和话术，与同门破冰的切入话题 |
| 🗓️ 推进时间线 | 6个月可落地计划：建立关系 → 验证假设 → 冲刺投稿 |
| 🏆 目标会议 | 每个 idea 对应目标会议（NeurIPS/ICML/ICLR/ACL）和预计周期 |
| 🌱 关系维护 | 预期摩擦点与化解方式，长期合作黏性提升策略 |

**核心价值：** 输出不是一份报告，而是一套**可以直接行动的合作剧本**。

<div align="center">

```
                    用户输入姓名

        │         OpenClaw Agent Framework        │

        │  ┌──────────┐  ┌──────────┐  ┌───────┐ │

        │  │ search   │  │  papers  │  │ news  │ │

        │         多源并行采集 (sessions_spawn)    │
        │                   │                     │
        │                   ▼                     │

        │         实体消歧 · 防幻觉核实           │
        │                   │                     │
        │                   ▼                     │
        │         精美 HTML 报告生成              │
        └─────────────────────────────────────────┘
                         │
                         ▼
              3分钟内完成 · 自动上传分享链接
```

</div>

**核心技术栈：**
- 🤖 **Claude Sonnet 4.6** via OpenRouter — 深度分析与报告生成
- ⚡ **OpenClaw Agent Framework** — 多工具编排与并行子 Agent
- 🔍 **wechat-article-search** — 微信公众号实时情报采集
- 📚 **arXiv API** — 论文核实与学术轨迹分析（逐篇验证，防同名混淆）
- 🛡️ **防幻觉机制** — 已确认 / 合理推断（标注）/ 未找到（如实说明）
- 🚀 **sessions_spawn** — 多人画像并行生成，效率翻倍
