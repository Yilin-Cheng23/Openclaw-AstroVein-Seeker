## 🌟 项目简介

**龙门星脉求索虾**是一套运行在 OpenClaw 平台上的 AI 学术关系智能体系统。

它不是一只帮你机械搜论文的虾，而是一只真正懂学术关系、懂科研成长、懂合作时机的智能求索虾。它把公开学术世界中的人物、论文、动态、关系与机会，组织成一张可计算的**学术星脉图**，让研究生不只看见信息，更看见方向、合作与未来。

### 🔭 能力1 · 导师向上管理 `MentorOS`
从微信公众号、arXiv、百度新闻、小红书等多源公开信息，构建导师的**近期基本面**——不是静态简历，而是动态的导师行动手册。3分钟生成，包含研究方向演变、性格风格洞察、关系网络图谱、具体沟通话术。

### 👥 能力2 · 同门近期画像 `PeerGraph`
重建同门的研究演变轨迹、技能图谱、合作风格与互补价值。多人**并行生成**，时间不翻倍。内置 arXiv 逐篇核实机制，防止同名混淆。

### 🚀 能力3 · 合作共赢未来 `Co-Lab Forge`
把导师支持、同门能力与用户方向整合成真正可执行的研究 idea、分工方案与推进路径。输出不是报告，而是**可以直接行动的合作剧本**。

**核心技术栈：**
- 🤖 **Claude Sonnet 4.6** via OpenRouter — 深度分析与报告生成
- ⚡ **OpenClaw Agent Framework** — 多工具编排与并行子 Agent
- 🔍 **wechat-article-search** — 微信公众号实时情报采集
- 📚 **arXiv API** — 论文核实与学术轨迹分析
- 🛡️ **防幻觉机制** — 已确认 / 合理推断（标注）/ 未找到（如实说明）
