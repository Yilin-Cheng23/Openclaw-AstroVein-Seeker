# 多源搜索策略

## 1. 微信公众号搜索

### 方法A：直接搜索学者姓名
```bash
# 用百度搜索微信公众号文章
web_fetch "https://www.baidu.com/s?wd=site:mp.weixin.qq.com [师兄姓名] 最新"
```

### 方法B：搜索机构公众号
```bash
# 搜索[所在学院]/研究院公众号
web_fetch "https://www.baidu.com/s?wd=中关村人工智能研究院 微信公众号"
web_fetch "https://www.baidu.com/s?wd=北京[所在学院] 微信公众号"
```

### 方法C：用 wechat-article 抓取
```bash
# 如果找到了公众号文章链接
python3 skills/wechat-article/scripts/wechat_article.py "https://mp.weixin.qq.com/s/xxx"
```

## 2. 小红书搜索

### 策略
- 小红书内容更非正式，可能展示学者的生活侧面、性格特点
- 搜索关键词：学者姓名 + "学术" / "科研" / "导师"

```bash
# 百度搜索小红书内容
web_fetch "https://www.baidu.com/s?wd=site:xiaohongshu.com [师兄姓名]"
```

---

## 3. arXiv 最新论文

### 策略
- 只看**最近3-6个月**的论文
- 不用历史论文判断当前关注点（论文有滞后性）

```bash
# 搜索最近的 arXiv 论文
web_fetch "https://arxiv.org/search/?query=Zheng+Shuxin&searchtype=author&order=-announced_date_first"
```

---

## 4. 百度新闻（实时动态）

### 策略
- 搜索最近的新闻、会议、活动、采访
- 加上时间限制（2025、2026）

```bash
# 搜索最新动态
web_fetch "https://www.baidu.com/s?wd=[师兄姓名] 中关村 2025 2026"
web_fetch "https://www.baidu.com/s?wd=[师兄姓名] AI商学院 创业"
```

## 6. 学术会议/机构官网

### 策略

```bash
# [所在学院]官网
web_fetch "https://www.baidu.com/s?wd=site:[学院域名] [师兄姓名]"
```

## 7. 多源验证流程

### Step 1: 初步搜索
1. 百度搜索学者姓名 + 机构
2. 提取关键信息（职位、最新动态、研究方向）

### Step 2: 验证关键信息
1. 如果发现"发布了某系统"：
   - ✅ 搜索该系统的官方新闻稿
   - ✅ 确认学者是主导还是参与者
   - ✅ 标注来源

2. 如果发现"最新论文"：
   - ✅ 去 arXiv 或期刊官网验证
   - ✅ 确认作者顺序（一作？共同作者？）
   - ✅ 确认发表时间（是否最新？）

### Step 3: 交叉验证
- 同一个信息至少从2个不同来源确认
- 如果来源单一，标注"待验证"

## 8. 信息时效性判断

### 高时效（优先使用）
- ✅ 微信公众号文章（最近3个月）
- ✅ 小红书内容（最近3个月）
- ✅ 百度新闻（2025-2026）
- ✅ arXiv 论文（最近6个月）

### 中等时效（参考）
- ⚠️ 会议论文（NeurIPS/ICML/ICLR，发表后6-12个月）
- ⚠️ 期刊论文（Nature/Science，发表后12-24个月）

### 低时效（仅作背景）
- ❌ 2年前的论文（不用于判断当前关注点）
- ❌ 简历/个人主页（可能未更新）

## 9. 性格推断策略

### 从公开言行推断
1. **讲话风格**：
   - 引用"打退堂鼓" → 诚实、反思型
   - 强调"产业需求" → 实干派

2. **团队活动**：
   - 组织黑客松 → 重视工程能力
   - 博士生从0到1做项目 → 培养实战派

3. **职业选择**：
   - 从学者转向创业 → 理想主义 + 冒险精神
   - 担任商学院联席院长 → 跨界思维

### 偏好学生类型推断
- 看他现在带什么样的学生
- 看他在讲话中强调什么品质
- 看他的项目需要什么能力

---

## 10. 资源分析策略

### 导师关系杠杆
- 导师是谁？导师有什么资源？
- 例：[导师的导师]（[顶级研究院]首席研究员）→ 微软算力、数据、人脉

### 机构资源
- 所属机构的定位、资源、生态
- 例：中关村两院 → AI商学院、产业联合实验室、创投网络

### 可能的项目机会
- 基于他的最新工作推断
- 例：OmniScientist → 需要产品迭代、用户研究、功能设计
- 例：AI商学院 → 需要课程开发、案例研究

---

**核心原则：实时性 > 历史性，洞察 > 表面，可操作 > 泛泛而谈**
